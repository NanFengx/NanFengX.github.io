多维数组的问题

<?php
$List = array
(
    array(12,100,96),
    array(3,60,59),
    array(23,110,100)
);
$i=1;
echo "$List[$i][1]";
?>

运行时会显示 Array to string conversion …… 这样的问题。而一维数组没有发生问题。

多维数组，下标不是简单的数值的数组需要用{}起来。或者用temp，或者不使用引号都可以。

//修改后
echo "{$List[$i][1]}";
echo $List[$i][1];
$tmp=$List[$i][1];
echo "$tmp";

Max Function(C)

C语言使用宏定义取最大值

使用宏定义取两个值中最大值是一个很简单的题目，随手写下：

#define MAX(x,y) ((x)>(y)?(x):(y))

这样写可以应付大多数的情况了，但在某些情况下还是会出现问题，比如说：

#include<stdio.h>
#define MAX(x,y) ((x)>(y)?(x):(y))
int main(){
    int i=4;
    int j=5;
    printf("max=%d，i=%d,j=%d",MAX(i++,j++),i,j);
}

输出的结果是：

max=6,i=5,j=7

这和我们想象的不太一样，j的值应该是6。我们把MAX(i++,j++)带入看下

printf("max=%d，i=%d,j=%d",((i++)>(j++)?(i++):(j++)),i,j);

我们可以看到，j++这条命令被执行了两次，导致j的值为7，为了解决这个问题，我们修改下MAX。

#define MAX(x,y) ({\
typeof(x) _x=x;\
typeof(y) _y=y;\
_x>_y?_x:_y;\
})

用typeof获取x,y的数据类型，用_x和_y暂时保存x,y的值。避免了上面出现的问题。又或者是：

#define MAX(x,y) ({\
typeof(x) _x=x;\
typeof(y) _y=y;\
(void)(&_x==&_y);\
_x>_y?_x:_y;\
})

抛出Warning解决类型不同的问题（不是很理解这句话(void)(&_x==&_y);）。

比较相同类型的值时，也可以这样写（虽然我感觉没有上面的好用）：

#define MAX(type,x,y) ({\
type _x=x;\
type _y=y;\
_x>_y?_x:_y;\
})

大概就是这样，关于宏定义取最大值的问题，欢迎与我讨论

Sever multi-user confliguration

服务器多用户配置

今天，为了让一个朋友科学上网，我修改了服务器配置，添加了多个端口，实现了让不同用户用同一个服务器上网的功能，本文使用的服务器为 CentOS 7 x64。

实现

打开PuTTY，填入IP，打开服务器，使用Xshell同理。

填写账号密码，登陆，输入命令，创建配置文件。

vi /etc/shadowsocks.json

按i键，将内容修改为

{
 "server":"0.0.0.0", 
"local_address": "127.0.0.1",
"local_port":1080,
"port_password": {
      "8001": "foobar1",    #端口号，密码
      "8002": "foobar2"
},
 "timeout":300,
"method":"aes-256-cfb",
 "fast_open": false
}

按ESC退出，输入:wq保存文件。

有的服务器有防火墙

firewall-cmd --    zone=public --add-port=8001/tcp --permanent

这里8001是你的端口号，根据你设置的端口进行修改。

如果你想查看已经开启的端口，输入：

firewall-cmd --list-ports

查看已开启的端口。

重启防火墙：

firewall-cmd --reload

查看端口号是否开启，运行命令：

firewall-cmd --query-port=8001/tcp

大功告成。

其它

我使用vultr的服务器进行科学上网，每月有500G流量，正常情况基本用不完。开启多端口后，让你的同学朋友们下载Shadowsocks，配置一下，就让他们能够享受科学上网了。
