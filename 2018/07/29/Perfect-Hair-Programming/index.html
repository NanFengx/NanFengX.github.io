多维数组的问题

<?php
$List = array
(
    array(12,100,96),
    array(3,60,59),
    array(23,110,100)
);
$i=1;
echo "$List[$i][1]";
?>

运行时会显示 Array to string conversion …… 这样的问题。而一维数组没有发生问题。

多维数组，下标不是简单的数值的数组需要用{}起来。或者用temp，或者不使用引号都可以。

//修改后
echo "{$List[$i][1]}";
echo $List[$i][1];
$tmp=$List[$i][1];
echo "$tmp";
